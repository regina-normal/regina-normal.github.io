Citing Regina

If you find Regina useful in your research, please consider citing it as you would any other paper that you use. A suggested form of reference is:

Benjamin A. Burton, Ryan Budney, William Pettersson, et al., Regina: Software for low-dimensional topology, http://regina-normal.github.io/, 1999–2020.

A BibTeX entry for this reference is included below. 

@misc{regina,
    author = {Benjamin A. Burton and Ryan Budney and William Pettersson and others},
    title = {Regina: Software for low-dimensional topology},
    howpublished = {{\tt http://\allowbreak regina-normal.\allowbreak
        github.\allowbreak io/}},
    year = {1999--2020}
}
