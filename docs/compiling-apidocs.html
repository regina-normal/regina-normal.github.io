API Documentation

You can access full API documentation by selecting Help→Python API Reference from the menu. This describes in detail the C++ objects, classes and methods that Regina makes available. You can also read this API documentation online at regina.sourceforge.net.

Don't be worried because the menu says “Python Reference”. The API documentation is really a C++ reference: Python users are simply granted access to the C++ library through an extra layer of Python bindings.
