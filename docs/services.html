Chapter 12. Bug Tracking and Mailing Lists

Regina is hosted at SourceForge, which provides the following services:

  • A bug tracking system, which also handles feature requests.

  • An announcements list <regina-announce@lists.sourceforge.net>. To receive important announcements (such as new releases of Regina), you can subscribe here. This list has extremely low traffic (only a couple of messages each year).
