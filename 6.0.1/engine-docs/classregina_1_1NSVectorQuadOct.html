Detailed Description

An almost normal surface vector using quad-oct coordinates.

If there are t tetrahedra in the underlying triangulation, there must be precisely 6t coordinates. The first six coordinates will be for the first tetrahedron, the next six for the second tetrahedron and so on. For each tetrahedron, the first three coordinates represent the number of quadrilateral discs of type 0, 1 and 2 (see NormalSurface::quads()), and the final three represent the number of octagonal discs of type 0, 1 and 2 (see NormalSurface::octs()).
