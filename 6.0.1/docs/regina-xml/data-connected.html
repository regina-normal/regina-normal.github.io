Name

connected — Is a normal surface connected?

Synopsis

Content Model 
connected ::=
Attributes
NameType
valueCDATA (required)

Description

A connected element stores whether or not a normal surface is connected.

Parents

The following elements contain connected: surface.

Children

Element connected has no children.

Attributes

value

Either T or F, according to whether the parent surface is connected or disconnected respectively.

Past versions of Regina (versions 4.92 and earlier) used true or 1 instead of T, used false or -1 instead of F, and also used unknown or 0 to indicate that this property was unknown.
