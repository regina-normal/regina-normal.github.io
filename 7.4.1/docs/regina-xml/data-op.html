Name

op — Replaced by attributes of filtercomb

Synopsis

Content Model 
op ::=
Attributes
NameType
typeCDATA (required)

Deprecated since Regina 7.0

Description

This element was used in the past to encode a single logical operation, as used by a combination filter to combine several other surface filters. Since Regina 7.0, this logical operation is now stored in the attributes of the corresponding filtercomb element instead.

Parents

The following elements contain op: filter (combination).

Children

Element op has no children.

Attributes

type

Represents the specific logical operation being described. This must be either and or or.
