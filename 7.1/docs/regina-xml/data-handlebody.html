Name

handlebody — Does a triangulation represent an orientable handlebody?

Synopsis

Content Model 
handlebody ::=
Attributes
NameType
valueNUMBER (required)

Added in Regina 7.1

Description

A handlebody element stores whether or not the parent triangulation is in fact a triangulation of an orientable handlebody, and if so, of which genus.

This element both replaces and generalises the deprecated threeball and solidtorus elements from Regina 7.0 and earlier (which correspond to orientable handlebodies of genus 0 and 1 respectively).

Parents

The following elements contain handlebody: tri.

Children

Element handlebody has no children.

Attributes

value

If the parent triangulation represents an orientable handlebody, then this attribute stores the genus as a non-negative integer. Otherwise this attribute stores the integer -1.
