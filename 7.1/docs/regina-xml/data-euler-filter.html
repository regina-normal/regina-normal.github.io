euler (filter)

PrevDeprecated and Removed Elements  Next

Name

euler (filter) — Replaced by attributes of filterprop

Synopsis

Content Model 
euler ::= (#PCDATA)

Deprecated since Regina 7.0

Description

This element was used in the past to represent a constraint on the Euler characteristic of a normal surface. Since Regina 7.0, this constraint is now stored in the attributes of the corresponding filterprop element instead.

An euler (filter) element encodes a list of allowed Euler characteristics in the character data for the XML element. This character data should contain a whitespace-separated list of integers; a surface will satisfy this constraint if its Euler characteristic belongs to this list.

The list of acceptable Euler characteristics must not be empty.

Parents

The following elements contain euler (in the context of a constraint on surface properties): filter (properties).

Children

Element euler (in the context of a constraint on surface properties) has no children.
