Detailed Description

template<class Derived, int dim>
class regina::alias::FacesOfTriangulation< Derived, dim, 1 >

Helper class that provides dimension-specific aliases for faces<0,1>().

This is inherited by the class Derived, which must provide a template function of the form T faces<subdim>() const, for facial dimensions subdim ≤ 1.

The return type T must be a const reference to the type detail::FaceListHolder<Derived, subdim>::Holder.
