Detailed Description

template<class Derived, int dim>
class regina::alias::FaceOfTriangulation< Derived, dim, 4 >

Helper class that provides dimension-specific aliases for countFaces<0,...,4>() and face<0,...,4>(size_t).

This is inherited by the class Derived, which must provide template functions of the form size_t countFaces<subdim>() and Face<dim, subdim>* face<subdim>(size_t) const, for all facial dimensions subdim ≤ 4.
