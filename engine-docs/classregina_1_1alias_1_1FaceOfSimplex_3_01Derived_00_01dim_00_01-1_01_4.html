template<class Derived, int dim>
class regina::alias::FaceOfSimplex< Derived, dim, -1 >

Helper class that provides nothing whatsoever.

This is an extreme case of FaceOfSimplex<Derived, dim, maxdim>, which provides dimension-specific aliases for face<subdim>(int) and faceMapping<subdim>(int), where reasonable, for facial dimensions subdim ≤ maxdim.
