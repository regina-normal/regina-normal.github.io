Detailed Description

An output iterator used to insert hypersurfaces into an NNormalHypersurfaceList.

Objects of type NNormalHypersurface* and NNormalHypersurfaceVector* can be assigned to this iterator. In the latter case, a surrounding NNormalHypersurface will be automatically created.
