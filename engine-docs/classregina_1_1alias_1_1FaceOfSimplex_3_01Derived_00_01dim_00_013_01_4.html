Detailed Description

template<class Derived, int dim>
class regina::alias::FaceOfSimplex< Derived, dim, 3 >

Helper class that provides dimension-specific aliases for face<0,...,3>(int) and faceMapping<0,...,3>(int).

This is inherited by the class Derived, which must provide template functions of the form Face<dim, subdim>* face<subdim>(int) const and Perm<dim+1> faceMapping<subdim>(int) const, for all facial dimensions subdim ≤ 3.
