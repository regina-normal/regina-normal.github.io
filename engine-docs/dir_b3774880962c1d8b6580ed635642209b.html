hypersurface Directory Reference

Files

file  hscoordregistry.h
 Provides access to a registry of all coordinate systems that can be used to create and store normal hypersurfaces in 4-manifold triangulations.
 
file  hypercoords.h
 Defines constants for normal hypersurface coordinate systems.
 
file  hyperflags.h
 Defines constants and flags for normal hypersurface enumeration.
 
file  nhsmirrored.h
 Provides a normal hypersurface vector that is mirrored in another coordinate system to avoid frequent lengthy calculations.
 
file  nhsstandard.h
 Implements normal hypersurface vectors using standard tetrahedron-prism coordinates.
 
file  nnormalhypersurface.h
 Deals with an individual normal hypersurface in a 4-manifold triangulation.
 
file  nnormalhypersurfacelist.h
 Contains a packet representing a collection of normal hypersurfaces in a 4-manifold triangulation.
 
file  nxmlhypersurfacereader.h
 Deals with parsing XML data for normal hypersurface lists.
