Standard dimensions

Whilst Regina was originally designed for working with 3-manifolds, it now offers varying levels of support for manifolds and triangulations in other dimensions.

Regina's standard dimensions are those for which it offers rich support (as opposed to basic support). For the current release, the standard dimensions are 2, 3 and 4.

This list is expected to grow in future releases.
