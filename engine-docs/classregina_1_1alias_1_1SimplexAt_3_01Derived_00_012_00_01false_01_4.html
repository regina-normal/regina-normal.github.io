template<class Derived>
class regina::alias::SimplexAt< Derived, 2, false >

Helper class that provides dimension-specific aliases for simplex(size_t).

This is inherited by the class Derived, which must provide a routine of the form Simplex<dim>* simplex(size_t) const.
