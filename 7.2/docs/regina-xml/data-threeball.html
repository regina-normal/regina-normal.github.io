Description

This element was replaced by the more general handlebody element in Regina 7.1. The old name threeball has been kept for backward compatibility, in that Regina still understands it when reading data files; however, any new code that writes data files should use handlebody instead.

A threeball element stores whether or not the parent triangulation is in fact a triangulation of the 3-dimensional ball.

Parents

The following elements contain threeball: tri.

Children

Element threeball has no children.

Attributes

value

Either T or F, according to whether the parent triangulation does or does not represent the 3-dimensional ball.
