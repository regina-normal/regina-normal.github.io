Deprecated and Removed Elements

The following XML elements were used in older Regina data files, but are no longer part of the current file format.

Those elements that are marked as deprecated can still be read by the current version of Regina, but will not be written when a data file is saved.

Those elements that are marked as removed can no longer be read or written. If Regina attempts to read an old data file containing removed element(s), those removed elements will be silently ignored.
