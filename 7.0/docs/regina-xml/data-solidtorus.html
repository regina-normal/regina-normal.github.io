Name

solidtorus — Does a triangulation represent the solid torus?

Synopsis

Content Model 
solidtorus ::=
Attributes
NameType
valueCDATA (required)

Description

A solidtorus element stores whether or not the parent triangulation is in fact a triangulation of the solid torus.

Parents

The following elements contain solidtorus: tri.

Children

Element solidtorus has no children.

Attributes

value

Either T or F, according to whether the parent triangulation does or does not represent the solid torus.
