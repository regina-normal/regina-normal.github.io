Regina for openSUSE Leap

Regina now offers openSUSE packages through online package repositories. This helps openSUSE to find and install dependencies automatically. It also helps with security, since the repository is cryptographically signed.

To install Regina and keep up-to-date with the newest version, follow these instructions.

You only need to follow these steps the first time you install Regina. After this, openSUSE's package manager will handle upgrades for you, just like for any other openSUSE package.

  1. Open a terminal (e.g., Konsole or GNOME Terminal).

  2. Add Regina to your list of package repositories. To do this, type the line below, replacing VERSION with your openSUSE Leap version (e.g., 15.6):

    sudo zypper addrepo https://people.debian.org/~bab/rpm/regina/opensuse/VERSION/regina.repo

    For example, if you are running openSUSE Leap 15.6, you would type:

    sudo zypper addrepo https://people.debian.org/~bab/rpm/regina/opensuse/15.6/regina.repo

  3. If you are running openSUSE Leap 15.6, you will need to import Regina's signing key manually. I'm sorry, openSUSE have caused so many problems over the years and this time they broke key imports for third-party repositories. Earlier versions of openSUSE worked fine, and the bug has been fixed so hopefully later versions will work fine also. On 15.6 however, you need to:

    • download Regina's signing key: https://people.debian.org/~bab/regina-key.txt

    • verify that the key you downloaded has the correct fingerprint (the output from the following gpg command should include the same 40-digit fingerprint 519A...38D9 that is shown at the bottom of this page):

      gpg --import-options show-only --import regina-key.txt

    • import this as an RPM signing key:

      sudo rpm --import regina-key.txt

  4. Now update the list of available packages. To do this, type:

    sudo zypper refresh

    Unless you already imported the key in the previous step, you will now be asked to import Regina's cryptographic signing key. Check that the fingerprint is correct (the correct fingerprint is shown at the bottom of this page), and then select trust always.

  5. Install Regina. To do this, type:

    sudo zypper install regina-normal

That's all! You can now keep Regina up to date in the same way that you manage your other openSUSE packages.
