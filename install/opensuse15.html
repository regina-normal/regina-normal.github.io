Regina for openSUSE Leap 15.x

Regina for openSUSE requires the libtokyocabinet9 library package. However, openSUSE stopped shipping this package with openSUSE Leap 15.0.

You will need to download libtokyocabinet9 manually from software.opensuse.org:

  • Visit software.opensuse.org;
  • Scroll down to your version of openSUSE (e.g., openSUSE Leap 15.0);
  • Click on “Show experimental packages”;
  • Find the version marked devel:libraries:c_c++;
  • Press the button marked “1 Click Install”.

Now you can install the Regina RPM in the normal way.
