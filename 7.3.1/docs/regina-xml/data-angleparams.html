Name

angleparams — Replaced by attributes of angles

Synopsis

Content Model 
angleparams ::=
Attributes
NameType
tautonlyCDATA (required)

Deprecated since Regina 7.0 / Added in Regina 4.90

Description

This element was used in the past to store the parameters for an enumeration of vertex angle structures. These are typically the parameters supplied by the user when creating a new angle structure list. Since Regina 7.0, these enumeration parameters are now stored in the attributes of the corresponding angles element instead.

Parents

The following elements contain angleparams: packet (angle structure list).

Children

Element angleparams has no children.

Attributes

tautonly

Either T if only taut structures are enumerated, or F if all vertex angle structures are enumerated.

Regina versions 4.6 and earlier did not write this attribute (and indeed, they omitted the entire angleparams element). If it is missing, it will be treated as F (since taut enumeration was not offered in those very old versions of Regina).
