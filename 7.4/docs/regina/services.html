Chapter 14. Bug Tracking and Release Announcements

Since November 2016, Regina is hosted at GitHub, which provides an issue tracking system for managing both bug reports and feature requests.

The old mailing list for announcements has now been decommissioned. Instead, you can subscribe to releases on GitHub:

  • Log into GitHub (you will need a GitHub account for this);

  • Visit Regina's GitHub repository and press Watch;

  • Choose Custom, and check the box to watch for releases.
