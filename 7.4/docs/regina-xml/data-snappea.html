snappea

Name

snappea — Replaced by snappeadata

Synopsis

Content Model 
snappea ::= (#PCDATA)

Deprecated since Regina 7.0

Description

This element was used in the past to store the contents of a SnapPea triangulation packet. Since Regina 7.0, SnapPea triangulations now store their content directly in the character data of the snappeadata element (which replaces the old packet element for SnapPea triangulations).

A snappea element contains the full contents of a SnapPea data file, using SnapPea's own text-based file format. Whitespace and line breaks are relevant and are considered a part of the data file.

Parents

The following elements contain snappea: packet (SnapPea triangulation).

Children

Element snappea has no children.
