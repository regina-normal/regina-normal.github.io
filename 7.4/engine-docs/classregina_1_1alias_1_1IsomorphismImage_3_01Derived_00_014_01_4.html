Detailed Description

template<class Derived>
class regina::alias::IsomorphismImage< Derived, 4 >

Helper class that provides dimension-specific aliases for both const and non-const versions of simpImage(size_t).

This is inherited by the class Derived, which must provide functions of the form int& simpImage(size_t) and int simpImage(size_t) const.
